Below is a list of JAVA projects completed for college classes. There is an executable jar file available for download.

  Contact List

The Contact List project was a group project for our final assignment of the semester. We were tasked with creating a Contact List that would store three types of contacts, Family, Friend and Business. We were also required to save the contact list as a serialized file and read the contents of the file when updating the registry. It needed to be sortable by contact type and searchable. Being able to add, update and delete contacts was also required.
